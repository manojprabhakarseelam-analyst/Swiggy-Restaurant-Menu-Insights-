# Swiggy-Restaurant-Menu-Insights
🍽️ Swiggy Restaurant & Menu Insights Dashboard

📌 Overview
This project focuses on analyzing a real-world Swiggy dataset to uncover insights about restaurant distribution, food preferences, and category trends across different cities. The goal was to clean and structure raw data, perform exploratory data analysis (EDA), and build an interactive dashboard to support data-driven decision-making.

#🛠️ Tools & Technologies
Python (Pandas, NumPy)
Excel
Power BI (DAX, Power Query)
Exploratory Data Analysis (EDA)
Data Visualization

📊 Key Objectives
Clean and standardize raw dataset to ensure data accuracy
Analyze city-wise restaurant distribution
Identify high-demand food categories across regions
Understand customer preferences (Veg vs Non-Veg trends)
Build an interactive dashboard for business insights

🔍 Key Insights
Identified top cities with highest restaurant concentration
Discovered popular food categories driving demand across regions
Revealed regional dietary patterns using Veg vs Non-Veg classification
Highlighted top-performing restaurants and category trends

📈 Dashboard Features
City-wise restaurant distribution analysis
Category-wise performance tracking
Veg vs Non-Veg trend visualization
Interactive filters for dynamic exploration
KPI tracking for quick business insights

⚙️ Project Workflow
Data Cleaning & Preprocessing
Exploratory Data Analysis (EDA)
Feature Engineering (classification, grouping)
Data Visualization & Dashboard Development
Insight Generation

🎯 Outcome
This project demonstrates the ability to transform raw data into structured insights, enabling better understanding of customer preferences and market trends in the food delivery domain.
